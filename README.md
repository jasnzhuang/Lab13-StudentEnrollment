# Lab13-StudentEnrollment
ASP.NET Core web application written in C#, which enables the creation and managing of a student enrollment system by the school administration.

Azure Deployment - [Link](http://paulstudentenrollment.azurewebsites.net/)

## Setup

1) Clone repo from GitHub into a directory of your choice
2) Locate directory where the repo was cloned into
3) Open the Lab13-StudentEnrollment directory
4) Click on MvcMovie.sln to open the program in Visual Studio
